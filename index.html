Setup Instructions:

  1. Create a Google Sheet with columns: Barcode, Timestamp, Image
  2. Go to Extensions → Apps Script
  3. Copy and paste the provided script code
  4. Deploy as Web App (Execute as: Me, Access: Anyone)
  5. Copy the Web App URL and paste it above
